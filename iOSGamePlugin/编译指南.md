# iOS游戏插件编译指南

## 📱 苹果手机无法直接编译

iOS设备没有编译环境，必须在macOS上编译。

## 🖥️ 编译方法

### 方法一：macOS本地编译（推荐）

#### 1. 环境准备
- macOS 10.15 或更高版本
- Xcode 12.0 或更高版本
- 安装Command Line Tools

#### 2. 安装Theos
```bash
# 打开终端，运行以下命令
bash -c "$(curl -fsSL https://raw.githubusercontent.com/theos/theos/master/bin/install-theos)"
```

#### 3. 编译插件
```bash
# 进入项目目录
cd iOSGamePlugin

# 编译
make

# 输出文件：.theos/obj/GamePlugin.dylib
```

### 方法二：GitHub Actions自动编译（无需mac电脑）

#### 1. 上传代码到GitHub
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/yourusername/iOSGamePlugin.git
git push -u origin main
```

#### 2. 自动编译
- 每次推送代码会自动触发编译
- 编译好的dylib文件会作为Artifacts下载
- 也可以手动触发工作流

#### 3. 下载编译结果
1. 进入GitHub仓库
2. 点击 Actions 标签
3. 选择最新的工作流运行
4. 下载 Artifacts 中的 GamePlugin.dylib

### 方法三：使用macOS虚拟机

#### 1. 虚拟机软件
- VMware Workstation
- VirtualBox
- Parallels Desktop（Mac）

#### 2. 安装macOS
- 需要macOS镜像文件
- 分配至少4GB内存
- 分配至少50GB硬盘空间

#### 3. 在虚拟机中编译
按照方法一的步骤在虚拟机中编译

### 方法四：租用云服务器

#### 1. 云服务商
- AWS EC2 (macOS实例)
- MacStadium
- Scaleway

#### 2. 连接服务器
```bash
ssh username@server-ip
```

#### 3. 在服务器上编译
按照方法一的步骤编译

## 📦 使用编译好的插件

### 1. 获取dylib文件
编译完成后，你会得到 `GamePlugin.dylib` 文件

### 2. 注入IPA
```bash
# 使用注入脚本
./inject_plugin.sh 游戏.ipa 游戏_修改版.ipa
```

### 3. 安装到iPhone
- 使用AltStore
- 使用Sideloadly
- 使用Xcode

## 🔧 常见问题

### Q: 没有Mac电脑怎么办？
A: 使用GitHub Actions自动编译，完全免费

### Q: 编译报错"theos not found"？
A: 确保正确安装Theos并设置环境变量

### Q: 编译报错"SDK not found"？
A: 安装Xcode并选择正确的SDK版本

### Q: 如何修改卡密？
A: 修改 GamePlugin.m 中的 validKeys 数组
